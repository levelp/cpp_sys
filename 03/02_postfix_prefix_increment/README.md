Две формы записи для инкремента и декремента
* Префиксная: ++i --i
* Постфиксная: i++ i--
Постфиксная форма -
сначала используем значение, потом увеличиваем
``` cpp
  int j = i++ + i++; // j = 4 + 4, а потом 2 раза увеличили i
  SHOW(i); // i = 8
  SHOW(j); // j = 6
```

Префиксная форма -
сначала увеличиваем i, потом присваиваем j
``` cpp
  i = 1;
  j = ++i;
  SHOW(i) // i = 2
  SHOW(j) // j = 2
```
