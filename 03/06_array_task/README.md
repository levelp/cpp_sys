Задание: сумма элементов массива
--------------------------------
Нумерация элементов массива с 0
Все элементы на чётных местах взять с "+"
Все элементы на нечётных местах с "-"
Чётные: 0, 2, 4, 6 и т.д.
Нечётные: 1, 3, 5...
Пример массива
``` cpp
  const int N = 6;
  int A[N] = {1, 3, -2, 4, 2, 10};
```

Пример вызова функции
``` cpp
  // 1 - 3 -2 -4 + 2 -10 = -16
  cout << sum(A, N) << endl;
```
