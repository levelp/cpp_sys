Условный оператор if
--------------------
Синтаксис:
``` cpp
  if(/*Условие*/ 1 )
    cout << "Если условие истинно!" << endl;
  else
    cout << "Если условие ложно!" << endl;

```

Всегда ложное условие
``` cpp
  if(0) {
    // Сюда мы никогда не попадём
  }

```

Всегда истинное условие
``` cpp
  if(1) {
    // Сюда мы попадём всегда
  }

```

Операции сравнения:
-------------------
* >= - больше или равно
* <= - меньше или равно
* != не равно
* !a
* > - больше
* < - меньше
* && - логическое И
* || - логическое ИЛИ
